6. JavaScript uitzetten

1. Problemen die deze features veroorzaken.

  • Functionaliteiten – Functionaliteiten van de website kunnen worden beperkt of volledig zijn uitgeschakeld.
  • Gebruikerservaring – De gebruikerservaring kan negatief worden beïnvloed, aangezien sommige functies en effect mogelijk niet werken, omdat deze afhankelijk zijn van JavaScript.

2. Hoe je dit zou kunnen testen.

  • Dev Tools – Je zou gebruik kunnen maken van de Developer Tools in je browser, om JavaScript uit te schakelen.
  • Extensies – Je zou extensies kunnen gebruiker om de JavaScript uit te schakelen.

3. Websites waar dit ook problemen oplevert.

  • Netflix – Het overzicht van films op Netflix ziet er minder spectaculair uit zonder JavaScript, maar het lijkt erop dat je nog steeds door de films kunt bladeren. Het is echter niet mogelijk om de films te bekijken zonder JavaScript, naar mijn mening.
  • Pinterest – Pinterest daarentegen werkt niet zonder JavaScript, vandaar dat je ook de tekst "Oh no! Pinterest doesn't work without JavaScript" te zien krijgt.

4. Hoe kunnen wij dit oplossen.

  • Progressive Enhancement – De basisfunctionaliteit van de website implementeren zonder JavaScript en vervolgens extra functionaliteiten en effect toevoegen met behulp van JavaScript.
  • Server- side rendering– Door gebruik te maken van server- side rendering om de inhoud van de website te genereren op de server in plaats van op de client. .
