2. Custom fonts

1. Problemen die deze features kunnen veroorzaken

  • - Traag laden van de website
  • - Site kan zonder text komen te zitten door slechte fallbacks
  • - fallback font kan niet lijken op de custom font

2. Hoe zou je dit kunnen testen

  • - Development tools van de browser kijken in hoevere het de perfomance beinvloed
  • - Met inspector de CSS class uitzetten om te kijken wat de fallback is.
  • - Custom fonts helemaal uitzetten

4. Hoe kunnen wij dit oplossen

  • - Een (gedeeltelijke) oplossing is het kiezen het juiste formaat van je font, zo is een .WOFF2 bestand gemiddeld al 75% kleiner dan een origineel .TTF bestand.
  • - @font-face gebruiken om fonts gedeeltelijk te pre-loaden.
  • - Zorg dat je een fallback lijkt op de font die je wilt gebruiken
